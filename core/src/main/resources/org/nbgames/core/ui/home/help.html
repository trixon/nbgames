nbGames

nbGames is a platform for desktop games. Games that are provided as plugins and can easily be added and removed.

Start with adding some players and after that, let the games begin.

Have fun!

Keyboard shortcuts

  • F1 Display help
  • F11 Toogle full screen mode
  • ctrl + h Home
  • ctrl + t toggle toolbar visibility
  • ctrl + p Players
  • ctrl + g Manage games
  • ctrl + , Options
  • ctrl + n New Game
  • ctrl + i Information
  • ctrl + q Quit
